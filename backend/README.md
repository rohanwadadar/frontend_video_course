# Student Progress Tracking Backend

## Tech Stack
- Java 17
- Spring Boot
- MySQL
- JPA / Hibernate

## Features
- Automatic progress tracking
- Video watch percentage
- Time spent tracking
- Auto status update (Not Started / In Progress / Completed)
- No manual progress update

## API
POST /api/progress/update

## Storage
- Videos & PDFs stored locally
- Only URLs saved in database

## Note
This backend focuses on real learning behavior tracking.
